เว็บนี้จะเป็นการสอนสำหรับภาษาluaพื้นฐาน

เหมาะสำหรับใครที่กำลังศึกษาภาษาluaเพื่อนำไปสร้างแมพroblox

คำศัพท์ คำอ่าน แปล
local โลเคิล ตัวแปรเฉพาะที่ใช้ในสคลิปนั้น
script สคลิป ไฟล์โค้ด
game เกม ตัวเกมหลักของroblox
workspace เวิร์กสเปช พื้นที่ทำงาน
parent แพเรนต์ ตัวแม่/ของที่ครอบอยู้
child ชายด์ ตัวลูก
function ฟังก์ชัน ชุดคำสั่ง
end เอนด์ จบ(ปิดfunction/if)
if อิฟ ถ้า
then เด็น แล้วทำสิ่งนี้
wait() เวท รอเวลา
prnt() พริ้นท์ แสดงข้อตวามใน output
require รีไควร์ เรียกModulescriptมาใช้
retum รีเทิร์น ส่งค่ากลับ
pairs แพร์ส วนลูปตาราง
true ทรู จริง
false ฟอลส์ เท็จ
nil นิล ไม่มีค่า
number นัมเบอร์ ตัวเลข
string สตริง ข้อความ
part พาร์ท ชิ้นส่วน
model โมเดล กลุ่มของpart
player เพลดยอร์ ผู้เล่น
Character แคแรคเตอร์ ตัวละครผุ้เล่น
Humanoid ฮิวมานอยด์ ระบบชีวิตของตัวละคร
Event อีเวนต์ เหตุการณ์
Connect คอนเน็ก เชื่อมเหตุการณ์
Touched ทัชด์ เมื่อโดนซน
MouseButton1Click เมาส์บัตตันวันคลิก คลิกเมาส์ซ้าย
KeyCode คีย์โค้ด ปุ่มบนคีย์บอร์ด
for ฟอร์ วนลูปทำซ้ำ
while ไวล์ ทำซ้ำตราบใดที่เงื่อนไขเป็นจริง
repeat รีพีท ทำซ้ำก่อนแล้วค่อยเช็คเงื่อนไข
until อันทิล จนกว่า
table เทเบิล กล่องเก็บข้อมูลหลายค่า
insert อิน-เสิร์ท เพิ่มข้อมูลเข้า table
remove รี-มูฟ ลบข้อมูลจาก table
GetService เก็ท-เซอร์-วิส เรียกบริการของ Roblox
Players เพลเยอร์ส ระบบผู้เล่น
UserInputService ยูส-เซอร์-อิน-พุท-เซอร์-วิส ระบบรับปุ่มคีย์บอร์ด/มือถือ
WaitForChild เวท-ฟอร์-ชายด์ รอจนกว่าจะเจอของลูก
FindFirstChild ไฟนด์-เฟิร์ส-ชายด์ หา ถ้ามี
Vector3 เวก-เตอร์-ทรี ค่าต าแหน่ง 3 มิติ (x y z)
CFrame ซี-เฟรม าแหน่ง + การหมุน
TweenService ทวีน-เซอร์-วิส ทำาอนิเมชันลื่นๆ
TweenInfo ทวีน-อิน-โฟ ข้อมูลการขยับ (เวลา รูปแบบ)
RemoteEvent รี-โมท-อีเวนต์ ส่งข้อมูลระหว่าง Client ↔ Server
FireServer ไฟร์-เซิร์ฟเวอร์ ส่งข้อมูลจากผู้เล่นไปเซิร์ฟเวอร์
FireClient ไฟร์-ไคลเอนต์ ส่งข้อมูลจากเซิร์ฟเวอร์ไปผู้เล่น
debounce ดี-บาวซ์ กันกดซ่ำ / กันทำงานถี่
ModuleScript โม-ดูล-สคริปต์ สคริปต์เก็บโค้ดเป็นระบบ(แยกโค้ดให้สะอาด)
require() รี-ไควร์ เรียก ModuleScript
ServerScriptService เซิร์ฟเวอร์-สคริปต์-เซอร์วิส ที่เก็บสคริปต์ฝั่งเซิร์ฟเวอร์
StarterPlayerScripts สตาร์ท-เตอร์-เพลเยอร์-สคริปต์ สคริปต์ฝั่งผู้เล่น
LocalScript โล-เคิล-สคริปต์ สคริปต์ฝั่งผู้เล่น
OnServerEvent ออน-เซิร์ฟเวอร์-อีเวนต์ รับข้อมูลจาก Client
OnClientEvent ออน-ไคลเอนต์-อีเวนต์ รับข้อมูลจาก Server
RootJoint รูท-จอยน์ ข้อต่อหลักของตัวละคร
Motor6D มอ-เตอร์-ซิกซ์-ดี ข้อต่อที่ขยับได้ (แขน ขา)
Weld / WeldConstraint เวลด์ เชื่อม Part เข้าด้วยกัน
CurrentCamera เคอร์-เรนต์-คา-เม-รา กล้องของผู้เล่น
CameraType คา-เม-รา-ไทป์ ประเภทกล้อง
CameraSubject คา-เม-รา-ซับ-เจ็กต์ สิ่งที่กล้องติดตาม
ContextActionService คอน-เท็กซ์-แอค-ชัน-เซอร์-วิส ระบบปุ่มข้ามแพลตฟอร์ม
BindAction ไบน์-แอค-ชัน ผูกปุ่ม
ScreenGui สกรีน-กูอี UI หลัก
ImageButton อิมเมจ-บัต-ตัน ปุ่มรูปภาพ
UIGradient ยูไอ-เกรเดียนท์ ไล่สี UI
UIScale ยูไอ-สเกล ปรับขนาดตามจอ
DataStoreService ดาต้า-สโตร์-เซอร์-วิส ระบบเซฟข้อมูล
OrderedDataStore ออ-เดอร์-ด-ดาต้า-สโตร์ กระดานจัดอันดับ
Attributes แอท-ทริ-บิวท์ ค่าติดตัว Object
task.wait() ทาสก์-เวท wait เวอร์ชันใหม่ แม่นกว่า
task.spawn() ทาสก์-สปอว์น รันโค้ดแยก
pcall พี-คอล กัน Error
State สเตท สถานะ
FSM (Finite State Machine) เอฟ-เอส-เอ็ม ระบบเปลี่ยนสถานะ
Detection ดิ-เทค-ชัน การตรวจจับ
Distance ดิส-แทนซ์ ระยะห่าง
Target ทาร์-เก็ต เป้าหมาย
PathfindingService พาธ-ไฟน์-ดิง-เซอร์-วิส ระบบหาเส้นทาง
CreatePath ครี-เอท-พาธ สร้างเส้นทาง
ComputeAsync คอม-พิวต์-อะ-ซิงค์ คำนวณเส้นทาง
Waypoint เวย์-พอยท์ จุดที่ NPC ต้องเดินผ่าน
PathStatus พาธ-สเตตัส สถานะเส้นทาง (ส าเร็จ / ล้มเหลว)
